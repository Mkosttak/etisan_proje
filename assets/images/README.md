## Not

Şu anda uygulama görseller olmadan da çalışacaktır. Material Icons ve text-based placeholder'lar kullanılmaktadır.

Görseller eklendiğinde, bunları kodda şu şekilde kullanabilirsiniz:

```dart
Image.asset('assets/images/logo.png')
```
