# Assets / Icons

Bu klasöre özel SVG ikonlarını ekleyiniz.

## Önerilen İkonlar

### Meal Types
- **icon_normal.svg** - Normal yemek
- **icon_vegetarian.svg** - Vejetaryen
- **icon_vegan.svg** - Vegan
- **icon_glutenfree.svg** - Glutensiz

### Payment
- **icon_wallet.svg** - Cüzdan
- **icon_credit_card.svg** - Kredi kartı
- **icon_qr_code.svg** - QR kod

### Categories
- **icon_breakfast.svg** - Kahvaltı
- **icon_lunch.svg** - Öğle yemeği
- **icon_dinner.svg** - Akşam yemeği

## SVG İkon Kaynakları

- [Flaticon](https://www.flaticon.com/)
- [Iconify](https://iconify.design/)
- [Heroicons](https://heroicons.com/)
- [Feather Icons](https://feathericons.com/)

## Kullanım

```dart
import 'package:flutter_svg/flutter_svg.dart';

SvgPicture.asset('assets/icons/icon_wallet.svg')
```

## Not

Uygulama şu anda Material Icons kullanmaktadır. Özel ikonlar opsiyoneldir.
